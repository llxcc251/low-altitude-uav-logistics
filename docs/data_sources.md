# 数据来源说明

## 无人机参数：美团第四代无人机（M4）

| 参数 | 值 | 来源类型 | 说明 |
|------|----|---------|------|
| `max_payload_kg` | 2.5 kg | ✅ M4 公开数据 | 最大载重 |
| `max_speed_ms` | 20.0 m/s | ✅ M4 公开数据 | 最高飞行速度 |
| `max_range_km` | 10.0 km | ✅ M4 公开数据 | 满载最大配送距离 |
| `delivery_radius_km` | 5.0 km | ✅ M4 公开数据 | 配送半径 |
| `charge_rate_c` | 2C | ✅ M4 公开数据 | 充电倍率 |
| `service_time_window_min` | 15 min | ✅ M4 公开数据 | 3 km 半径内 15 分钟达 |
| `operational_altitude_m` | 120 m | ✅ M4 公开数据 | 飞行高度上限 |
| `weather_tolerance` | 中雨/中雪/7级风 | ✅ M4 公开数据 | 天气适应性 |
| `cruise_speed_ms` | 15.0 m/s | ⚠️ 工程估算 | 最高速 20 m/s 的 75%，巡航合理值 |
| `battery_capacity_wh` | 1000 Wh | ⚠️ 工程估算 | 12S LiPo × 2 推算（需确认） |
| `max_flight_time_min` | 11.0 min | ⚠️ 工程估算 | 15 m/s 飞 10 km 推算 |
| `enable_cost` | 100 | 🔧 模型超参数 | 无人机启用固定成本，可调参 |
| `split_penalty` | 50 | 🔧 模型超参数 | 订单拆分惩罚系数，可调参 |

### 来源类型说明

- **✅ M4 公开数据**：来自美团官方公开的技术资料或报道
- **⚠️ 工程估算**：M4 未公开该参数，根据已知参数和工程经验推算
- **🔧 模型超参数**：与 M4 无关，人为设定的模型权衡系数，后期可调整
