# Python 版本

## 文件说明

| 文件 | 说明 |
|------|------|
| `config.py` | 参数配置 |
| `load_data.py` | 加载校区节点和订单数据 |
| `eval_solution.py` | 统一严格评估函数 |
| `random_search.py` | 随机搜索算法 |
| `genetic_algorithm.py` | 遗传算法 |
| `run_all.py` | 算法对比主脚本 |

## 运行方式

```bash
cd xphwork/python
python run_all.py
```

## 依赖

- Python 3.7+
- numpy

```bash
pip install numpy
```
