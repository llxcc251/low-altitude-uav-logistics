# 低空物流服务优化：多无人机协同调度

中山大学深圳校区 · 运筹学大作业

## 项目概述

以中山大学深圳校区为应用场景，研究多订单、多约束条件下的无人机协同调度问题。通过数学建模和优化算法，实现订单拆分、任务分配与路径规划，并开发前端可视化看板展示求解结果。

## 项目结构

```
src/
├── model/           # 数学模型定义（变量、约束、目标函数）
├── algorithms/      # 优化算法（贪心、模拟退火、节约算法等）
├── preprocessing/   # 数据预处理（订单拆分、路线生成）
└── utils/           # 工具函数

data/
├── raw/             # 原始输入数据
└── processed/       # 预处理后数据

frontend/            # 交互式答辩看板
├── public/
└── src/

docs/                # 文档
assets/              # 静态资源
```

## 方法

- 贪心合单
- 模拟退火
- 节约算法
- 贪心指派
- 最邻近法
- 单任务逐趟
- 随机策略
